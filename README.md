## About

tapScan stands for I2T (Image-to-Text) is an application that lets you convert text image into an editable and searchable text.

## Top Features

1. Auto Border Detection: The app automatically detects the borders of the document and crops it accordingly, saving time and effort.

2. OCR (Optical Character Recognition): The app uses OCR technology to recognize text in images and convert it into editable text.

3. PDF Maker: The app can convert any scanned document into a PDF file.
